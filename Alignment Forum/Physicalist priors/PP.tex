% Title: Physicalist priors: a formal theory of naturalized induction
 
\textbf{TLDR:}\Co{b} Solomonoff's theory of induction requires us to describe hypotheses about reality from the subjective point of view of the agent. So does conventional reinforcement learning theory. For such a hypothesis to be extremely precise, it is insufficient for it to contain the fundamental laws of physics, but it must also contain "[bridge rules](https://www.lesswrong.com/posts/ethRJh2E7mSSjzCay/building-phenomenological-bridges)" that enable locating the agent within the universe. These bridge rules are bound to be extremely complex, somewhat undermining the Occam's razor principle which Solomonoff's prior strives to implement. This creates multiple problems for Cartesian agents including the inability to exploit origin theories and bias to privileging oneself point of view. It also might allow [acausal attackers](https://ordinaryideas.wordpress.com/2016/11/30/what-does-the-universal-prior-actually-look-like/) to completely dominate an AI's prior with malign hypotheses. To solve these issues, we propose "physicalist" priors which don't require bridge rules. Instead, we go from the fundamental / microscopic / objective description of physics to the subjective / agent-centric description by detecting entanglement between uncertainty about physics and uncertainty about the output of the agents own source code. This is made entirely rigorous via infra-Bayesianism and Dempster-Shafer theory.

\begin{Huge}Introduction\end{Huge}

The standard model of perfect instrumental rationality is Bayesian reinforcement learning. We [challenged](https://www.alignmentforum.org/s/CmrW8fCmSLK7E25sa) this model before due to its problems with non-realizability, suggesting infra-Bayesianism as an alternative. In any case, both formalisms crucially depend on the choice of a prior. What is the "correct" prior? We consider it likely that there is no more a unique objective answer than there is a unique objective utility function. However, certain qualitative features of the prior seem to be objectively required, and some quantitative features are necessary for it to be competitive in our universe.

One principle we can be certain a reasonable prior has to embody is Occam's razor: the preference of simple hypotheses over complex hypotheses. A natural formalization of this principle is provided by the Solomonoff prior[\^{}1]:
